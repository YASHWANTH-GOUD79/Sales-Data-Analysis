# Sales Data Analysis Project

##  Project Overview

This project focuses on analyzing sales data to extract meaningful business insights using **SQL** and **Python**. The analysis helps understand sales performance across regions, products, and time periods, which supports better decision-making.

This is my **first Data Analyst project**, built to demonstrate practical data analysis skills expected from an entry-level Data Analyst.


##  Project Objectives

* Analyze overall sales performance
* Identify top-selling products
* Analyze region-wise revenue distribution
* Understand monthly sales trends
* Practice real-world data cleaning and analysis


##  Dataset Information

* **Dataset Name:** Superstore Sales Dataset
* **Source:** Kaggle
* **Format:** CSV
* **Data Includes:**

  * Order ID
  * Order Date
  * Product Name
  * Category
  * Region
  * Sales Amount
  * Quantity Sold

## Tools & Technologies Used

* **SQL (MySQL)** – Data extraction and aggregation
* **Python** – Data cleaning and analysis

  * Pandas
  * NumPy
  * Matplotlib
* **Excel** – Basic data review (optional)

##  Analysis Performed

### SQL Analysis

* Calculated total sales
* Performed region-wise sales analysis
* Identified top 5 products by revenue
* Analyzed monthly sales trends

### Python Analysis

* Loaded and explored the dataset
* Cleaned data by removing duplicates and handling missing values
* Performed group-based analysis using Pandas
* Created visualizations for better understanding of sales patterns


##  Key Insights

* Certain regions contributed significantly higher revenue compared to others
* A small number of products generated the majority of total sales
* Sales showed noticeable growth during specific months
* Product category had a strong influence on revenue
